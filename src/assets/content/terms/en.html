2. Service Description

Bam! is a peer-to-peer (P2P) file sharing platform that enables direct file transfer between browsers via WebRTC technology. Files are never uploaded to cloud servers: the transfer occurs exclusively in direct mode between the users' devices.

The service is available as a browser-accessible web application and as a desktop application via Electron, for Windows, macOS and Linux operating systems.

7. Encryption & Security

All transfers on Bam! are protected by native end-to-end encryption (E2EE), guaranteed by the DTLS protocol integrated in WebRTC. This means that data in transit is encrypted and cannot be intercepted by third parties.

  • Optional AES-GCM encryption: available for Pro Premium users, allows adding an extra layer of protection via password with PBKDF2 key derivation.
  • Integrity verification: SHA-256 hash to ensure received files are identical to those sent.

Bam! has no technical ability to access the content of files transferred between users. The P2P nature of the service ensures that no data transits through our servers.
